## 1. Обзор и Vision

**Somnia StarForge** — полностью on-chain auto-battler в стилистике «Эхо Снов» Somnia.  
Игроки собирают флот из NFT-кораблей и пилотов, расставляют их на поле и запускают детерминистичные бои, которые разрешаются on-chain.

**Ключевые достижения v1.1:**
- Полностью рабочий core loop
- Разбивка фронтенда на сцены (BootScene + PrepareScene + BattleScene)
- Hover tooltip на всех юнитах
- Подсветка team-slots + защита от stacking
- Плавная анимация появления новых юнитов
- Автообновление UI после всех действий
- Улучшенный UX при ошибках и очистка временных текстов

**Жанр:** Auto-Battler / Auto-Chess lite  
**Платформа:** Браузер (Phaser 3.90.0 + Vite) + Somnia Testnet/Mainnet

## 2. Core Gameplay Loop (v1.1 — полностью реализовано)

1. **Shop Phase**  
   - 5 слотов shop preview  
   - `rerollShop()` (0.0005 STT)  
   - `buyFromShop(uint256 slot)` (0.001 STT)

2. **Position Phase**  
   - Динамический owned units grid (8 колонок, компактный)  
   - Drag-and-drop на 4×2 grid  
   - Hover tooltip + подсветка свободных/занятых слотов

3. **Battle Phase**  
   - `startMatch(uint256[] team)`  
   - Отдельная сцена `BattleScene` с улучшенным visual replay

4. **Reward Phase**  
   - Автоматический mint наград  
   - Автообновление owned grid, shop и профиля

## 8. Roadmap

**v1.0** — StarForge MVP (готово)  
**v1.1** — Полировка UX и стабильность (завершена 26 апреля 2026)

**v1.2** — Постоянный ИИ-противник и прогрессия  
- Постоянный ИИ-оппонент для каждого игрока (одна фиксированная команда до победы)  
- Генерация новой команды ИИ только после победы над текущей  
- Глобальный уровень ИИ соответствует уровню игрока (те же баффы)  
- Прогрессия через поражения: игрок получает XP и уровень даже при проигрыше  
- Начальный противник — 4 слабых Common-юнита  
- Игрок постепенно «перерастает» ИИ за счёт прокачки уровня и глобальных баффов

**v1.3** — Баланс боя и система артефактов  
- Разработка и настройка баланса боёв (учёт уровня, синергий, глобальных баффов)  
- Введение **артефактов / реликвий** — усиления для отдельных кораблей или всего войска  
- Артефакты могут выпадать в наградах и давать постоянные или временные бонусы  
- Улучшенная система наград и экономического прогресса

**v1.5** — Ranked + Sequence gasless  
**v2.0** — Deep Space (новые фракции, торговля, улучшенный replay)  
**v3.0** — Somnia Void (4X-стратегия в галактике)
